About The Creators

My name is Zahra Hutchinson, and I am a highschool senior at ESUMS. This is my first game creation, which I am making for my New Media and Tech for Social Change (Capstone) class. I really enjoy making games, so this project is a perfect fit.

What is a game

In my opinion, a game is something that has a set of rules that can be played by yourself or with a group of people. I'd like to think that games are fun, but they aren't always for everyone.
